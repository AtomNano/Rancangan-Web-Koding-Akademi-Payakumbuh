## 3. Memulai Aplikasi

### 3.1 Mengakses Sistem
1. Buka browser dan akses URL sistem e-learning
2. Anda akan melihat **Landing Page** dengan informasi tentang Coding Akademi

### 3.2 Login ke Sistem
1. Klik tombol **"Masuk"** di pojok kanan atas
2. Masukkan **Email** dan **Kata Sandi** Anda
3. Klik tombol **"Login"**
4. Sistem akan mengarahkan Anda ke **Dashboard** sesuai peran

### 3.3 Lupa Kata Sandi
1. Di halaman login, klik **"Lupa Kata Sandi?"**
2. Masukkan alamat email terdaftar
3. Cek email Anda untuk link reset password
4. Klik link dan buat kata sandi baru

### 3.4 Logout dari Sistem
1. Klik nama profil Anda di pojok kanan atas
2. Pilih **"Logout"**
3. Anda akan diarahkan kembali ke halaman login

## 7. FAQ & Troubleshooting

### 7.1 Pertanyaan Umum

**Q: Saya lupa kata sandi, bagaimana cara reset?**
> Klik "Lupa Kata Sandi?" di halaman login, masukkan email, dan ikuti instruksi di email yang dikirim.

**Q: Mengapa materi yang saya upload tidak tampil ke siswa?**
> Materi harus diverifikasi oleh Admin terlebih dahulu. Cek status materi di menu "Materi Saya".

**Q: Bagaimana cara mengganti email?**
> Hubungi Admin untuk mengubah email akun Anda.

**Q: Apakah progress saya tersimpan jika browser tertutup?**
> Ya, progress membaca PDF dan video tersimpan otomatis.

### 7.2 Masalah Umum

#### Tidak Bisa Login
1. Pastikan email dan kata sandi benar (perhatikan huruf besar/kecil)
2. Cek Caps Lock keyboard
3. Coba reset kata sandi
4. Hubungi Admin jika akun mungkin dinonaktifkan

#### Materi Tidak Bisa Dibuka
1. Pastikan koneksi internet stabil
2. Coba refresh halaman (F5)
3. Clear cache browser
4. Coba browser lain

#### File Tidak Bisa Diupload
1. Cek ukuran file (maksimal 50 MB)
2. Pastikan format file didukung (PDF, MP4, WebM)
3. Coba kompres file sebelum upload

#### Halaman Error / Tidak Tampil
1. Refresh halaman (F5 atau Ctrl+R)
2. Clear cache browser (Ctrl+Shift+Delete)
3. Coba logout dan login kembali
4. Hubungi Admin jika masalah berlanjut
