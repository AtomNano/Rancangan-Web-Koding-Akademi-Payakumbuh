1. Pendahuluan

1.1 Tentang Sistem

Platform E-Learning "Materi Online" adalah sistem pembelajaran digital yang dirancang untuk mendukung proses belajar mengajar di Coding Akademi Payakumbuh. Sistem ini memfasilitasi interaksi antara tiga peran utama: Admin, Guru, dan Siswa.

‚ÑπÔ∏è Informasi: Sistem ini dibangun menggunakan framework Laravel dengan antarmuka modern dan responsif untuk pengalaman pengguna yang optimal.

1.2 Fitur Utama

üë• Manajemen Pengguna

Pendaftaran, pengelolaan, dan pengaturan hak akses untuk Admin, Guru, dan Siswa.

üìö Manajemen Kelas

Pembuatan kelas, penugasan guru, dan pendaftaran siswa ke kelas.

üìÑ Materi Pembelajaran

Upload materi PDF dan video dengan sistem verifikasi oleh admin.

üìä Tracking Progress

Pemantauan kemajuan belajar siswa secara real-time dengan progress bar visual.

‚úÖ Presensi Digital

Pencatatan kehadiran siswa per pertemuan dengan berbagai status.

üì• Export & Backup

Export laporan ke Excel dan backup database untuk keamanan data.

  • Unenroll tidak menghapus data siswa, hanya memutuskan hubungan dengan kelas

7. FAQ & Troubleshooting

7.1 Pertanyaan Umum (FAQ)

Q: Saya lupa kata sandi, bagaimana cara reset?

A: Klik "Lupa Kata Sandi?" di halaman login, masukkan email, dan ikuti instruksi di email yang dikirim.

Q: Mengapa materi yang saya upload tidak tampil ke siswa?

A: Materi harus diverifikasi oleh Admin terlebih dahulu. Cek status materi di menu "Materi Saya".

Q: Bagaimana cara mengganti email?

A: Hubungi Admin untuk mengubah email akun Anda.

Q: Apakah progress saya tersimpan jika browser tertutup?

A: Ya, progress membaca PDF dan video tersimpan otomatis.

7.2 Masalah Umum

Tidak Bisa Login

  1. Pastikan email dan kata sandi benar (perhatikan huruf besar/kecil)
  2. Cek Caps Lock keyboard
  3. Coba reset kata sandi
  4. Hubungi Admin jika akun mungkin dinonaktifkan

Materi Tidak Bisa Dibuka

  1. Pastikan koneksi internet stabil
  2. Coba refresh halaman (F5)
  3. Clear cache browser
  4. Coba browser lain

File Tidak Bisa Diupload

  1. Cek ukuran file (maksimal 50 MB)
  2. Pastikan format file didukung (PDF, MP4, WebM)
  3. Coba kompres file sebelum upload

Halaman Error / Tidak Tampil

  1. Refresh halaman (F5 atau Ctrl+R)
  2. Clear cache browser (Ctrl+Shift+Delete)
  3. Coba logout dan login kembali
  4. Hubungi Admin jika masalah berlanjut

Progress Tidak Terupdate

  1. Pastikan Anda sudah menyelesaikan materi (scroll ke halaman terakhir untuk PDF)
  2. Refresh halaman untuk melihat update terbaru
  3. Logout dan login kembali
  4. Hubungi Admin jika masalah berlanjut

Notifikasi Tidak Muncul

  1. Periksa ikon notifikasi (üîî) di navbar
  2. Refresh halaman
  3. Periksa pengaturan browser (pastikan notifikasi tidak diblokir)
